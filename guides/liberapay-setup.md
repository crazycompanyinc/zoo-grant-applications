# Liberapay — Setup Guide for ZOO

## Overview
**Platform:** Liberapay
**URL:** https://liberapay.com
**Type:** Recurring donations platform
**Fees:** 0% (payment processor fees only)
**Supports:** Teams, multiple currencies, integrations
**Activity:** $25K+ transferred weekly, 1,840 new accounts/month

## Why Liberapay for ZOO?
- **Zero platform fees** — Only payment processor fees (Stripe/PayPal)
- **Recurring donations** — Stable, predictable income
- **Team support** — Create a ZOO team with multiple members
- **Open source friendly** — Designed for open-source projects
- **No special requirements** — Just an email to create account

## Setup Steps

### Step 1: Create Account
1. Go to https://liberapay.com
2. Click "Log In or Create Account"
3. Enter email address
4. Confirm email
5. Set password

### Step 2: Create ZOO Profile
1. Set username: `zoo` or `zoo-nous`
2. Set display name: "ZOO — Open Source AI"
3. Add description: "Building open-source AI infrastructure and agent platforms"
4. Add avatar: ZOO logo
5. Add website: https://github.com/crazycompanyinc/zoo-website
6. Add GitHub: https://github.com/crazycompanyinc

### Step 3: Configure Payment Processing
1. Connect Stripe account (for credit card payments)
2. Connect PayPal account (for PayPal payments)
3. Set up bank account for payouts

### Step 4: Set Up Team (Optional)
1. Go to Teams dashboard
2. Create team: "ZOO"
3. Add team members
4. Configure revenue sharing

### Step 5: Set Donation Tiers
| Tier | Weekly Amount | Benefits |
|------|--------------|----------|
| Supporter | $1/week | Name on sponsors page |
| Fan | $5/week | Above + Discord role |
| Sponsor | $10/week | Above + logo on README |
| Champion | $25/week | Above + priority support |
| Partner | $50/week | Above + advisory call |

### Step 6: Promote
- Add Liberapay badge to all GitHub repos
- Add donation link to website
- Share on social media
- Include in newsletter
